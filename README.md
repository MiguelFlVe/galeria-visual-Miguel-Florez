# Resumen

## Descripción
La presente página es el resultado de la sección de "Transferencia" de la guía "Propiedades visuales". En esta, se utilizaron imágenes de los personajes de Dragon Ball, los cuales fueron presentados en una caja individual para cada uno. En todos los elementos de la página se utilizaron selectores CSS para ajustar su visualización a la deseada.

## ¿Cómo verlo en modo local?
Para visualizar la página de modo local, y trabajar sobre ella, es recomendado clonar el presente repositorio en el dispositivo donde se desea revisar, utilizando la herramienta git. Una vez clonado, es necesario abrir el repositorio en un visualidar de código, como lo es Visual Studio Code.
